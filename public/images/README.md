# Logo Files

## Logo cần lưu

Hãy lưu logo vàng "ROYAL WEDDINGS" mà bạn có thành:
- **File name**: `logo-royal.png` (hoặc `.jpg`, `.webp`)
- **Location**: Thư mục này (`public/images/`)

## Thông tin kỹ thuật

- **Kích thước khuyến nghị**: 120x48px (hoặc tỷ lệ tương tự)
- **Format**: PNG, JPG, hoặc WebP
- **Nền**: Nên có nền trong suốt hoặc nền trắng

## Các tệp đã cập nhật

Code đã được cập nhật để sử dụng logo mới tại các vị trí:
1. `components/auth/AuthMehappyShell.tsx` - Header desktop & mobile
2. `components/dashboard/DashboardChrome.tsx` - Dashboard header  
3. `components/landing/MarketingNavbar.tsx` - Marketing page header
